## Getting Started
1. Clona el repositorio y entra a la carpeta del proyecto.
2. Copia `.env.example` a `.env` e incluye tus credenciales.
3. Instala dependencias con `npm install` (o el gestor correspondiente).
4. Ejecuta `npm run dev` para un entorno local de desarrollo.

### Scripts comunes
- `npm run dev`: levanta el servidor de desarrollo con recarga en caliente.
- `npm run build`: genera la versión optimizada para producción.
- `npm test`: ejecuta el suite de pruebas automatizadas.
